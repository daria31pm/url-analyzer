# URL Analyzer - Анализатор подозрительных ссылок

Интеллектуальный анализатор URL для выявления фишинговых и вредоносных ссылок на основе машинного обучения.

## 🌟 Возможности

- **Многофакторный анализ** URL: длина, структура домена, наличие IP, подозрительные слова
- **Два типа моделей**: Random Forest и XGBoost
- **API с веб-интерфейсом** для удобного использования
- **Понятные объяснения** решений модели
- **Поддержка пакетного анализа** до 100 URL за запрос

## 🚀 Быстрый старт

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/url-analyzer.git
cd url-analyzer

# Установка зависимостей
pip install -r requirements.txt

# Создание примеров данных и обучение модели
python train.py --create-sample --samples 2000

# Запуск сервера
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
